    Projects

    Mise En Place

    Link Github

    Mise En Place is a web app that generates recipes from your fridge ingredients, reducing waste. It offers over 5,000 recipes, compiles favorites, and enables quick searches. It includes advanced filters for cuisine, meal type, intolerance, and diet.

    The frontend uses React, JavaScript, and Material-UI, while the backend utilizes Python, DjangoREST Framework, and SQLite for the database.

    Django Real Estate

    Link Github

    Django Real Estate is a feature-rich platform that allows users to search for real estate properties, connect with agents, and leverage advanced filtering options to find their dream home. It includes an admin page for complete control over property listings.

    The frontend is built using HTML/CSS, while the backend utilizes Django, Python, and PostgresSQL as the database.

    Smart Bills

    Link Github

    Smart Bills is an web application that calculates the hosts' bills based on split by dishes or split by headcount, then sends email to the hosts' friends based on the calculation.

    This is a project utilizes Node.js, React.js, Axios, and Material-UI for the frontend and UI design. JavaScript, MongoDB, and Express for backend and database.
